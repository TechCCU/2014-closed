TechCCU Workshop

2014

簡介

TechCCU 是由中正大學資工系學生@brucehsu與@Terces所發起，我們秉持技術討論與經驗分享的精神連結校內外各式的人才與我們一起交流。 過去一年來我們採用各種方式，如線上分享、實體聚會（Meet up）的方式進行我們的活動，隨著我們人數的增加，我們聚集到更多願意一起分享 的人來參與，眼見時機成熟，我們決定主辦一場為期一天的 Workshop 來讓大家共襄盛舉。

Skymizer 雲端編譯 / Skymizer: Build Software Better and Faster On The Cloud

源碼的效能，如工程師的熱血，沒有極限，等待你來引爆。編譯器有許多的限制，壓抑了源碼的效能。從源碼結構、到執行時期、到硬體架構都有限制。本次主題會從 GCC 編譯器的限制出發，介紹 Skymizer 中新穎與重要的最佳化技術，如雲端編譯與 SkyPat 分析工具，如何突破這些限制來加速源碼。希望藉由這次的機會，分享過去開發編譯器的經驗，和與會的朋友們一起成長。

本Session為台灣資安發展協會贊助，希望透過有系統的教學方式，讓學員們可以了解War Game的精神以及闖關技巧，並學習以及實際操作駭客的攻擊手法，寓教於樂。
